# Doctor's Portal 
## A Clinical Appointment Based Web Application

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Features

* Login
* Registration
* Appointment Booking
* Edit Appointment
* Delete Appointment
* Responsive Design
* Attractive Web Look

## Technology Used

* React
* React Router
* Firebase
* API
* Express JS
* Mongo DB
* Tailwind
* Daisy UI
